## Возможности
* Музыка с Вашей страницы
* Музыка друзей
* Музыка групп
* Рекомендации
* Поиск музыки, артистов
* Жанры
* Сохраненные
* Смена темы

## Минусы
* В данный момент сломано воспроизведение треков


Проект почти полностью написан на Kotlin
